# PixSearch

A simple and clean React application that allows users to search for images using the **Pixabay API**, browse paginated results, and view larger versions of images in a modal popup.

---

## Live Demo (Netlify Deployment)

**Live App:** [PixSearch] (https://phenomenal-sopapillas-eb4a40.netlify.app)

---

## Features

- Search images in real-time using Pixabay API  
- Responsive image grid layout  
- Pagination (Prev / Next)  
- Click image to view full-size modal  
- Mobile-friendly design  
- Fast and smooth UI  

---

## Technologies Used

- **React**
- **JavaScript (ES6+)**
- **Vite**
- **Pixabay API**
- **CSS Grid**
